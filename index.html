Jobseek

Jobseek is a Trello-style Kanban board designed to keep track of the basic details and status of the user's job applications. Cards may be dragged and dropped into any of the lists for a smooth user experience.

Built with: CSS3, React, Node.js, Express, PostgreSQL, Material-UI, and react-beautiful-dnd.

Live GitHub

Boba Finder

Boba Finder is an application born out of the question, "Wanna go get boba?" This application uses the Google Maps Places API and populates a list of boba stores in a given city.

Built with: HTML5, CSS3, vanilla JavaScript(ES6) and jQuery.

Live GitHub

SpeedTyper

SpeedTyper is a typing game built to test the number of words you can type in ten seconds. Styling was inspired by old-school 8-bit games.

Built with: React and CSS3.

Live GitHub
